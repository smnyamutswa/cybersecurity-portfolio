# Resume Project Bullets

## Automated Hybrid SOC Incident Detection and Response Platform

- Built a hybrid SOC pipeline integrating AWS-hosted Wazuh, Shuffle, TheHive, and Cortex with Windows, Ubuntu, and Kali Linux endpoints for automated detection, alert creation, threat-intelligence enrichment, and incident response.
- Simulated credential attacks, suspicious PowerShell activity, privilege escalation, and unauthorized authentication, then documented alerts, investigation timelines, MITRE ATT&CK mappings, and containment actions.

## Hybrid Identity Risk Analyzer

- Developed an agentic identity-security platform that analyzes Active Directory and AWS IAM identities to detect excessive permissions, privileged accounts, stale credentials, and insecure identity configurations.
- Built Python agents for identity collection, correlation, risk scoring, AI-generated investigation, remediation guidance, and Streamlit visualization.

## AI Vulnerability Management and Automated Remediation Platform

- Built a vulnerability-management workflow integrating OpenVAS, ServiceNow, and Python automation to prioritize findings, require analyst approval, remediate Linux, Windows, and Active Directory issues, verify fixes, and update tickets.
- Implemented platform-specific remediation dispatching, post-remediation verification, and end-to-end audit tracking.

## Secure AWS E-Commerce Platform

- Deployed a highly available AWS e-commerce platform using CloudFront, WAF, ALB, Auto Scaling EC2, RDS PostgreSQL, S3, and Route 53 across public and private subnets.
- Implemented layered monitoring and security with GuardDuty, Security Hub, CloudTrail, CloudWatch, AWS Config, and GitHub Actions CI/CD.

## Wazuh–Shuffle–TheHive–Cortex Incident Response Lab

- Automated Wazuh alert delivery through Shuffle into TheHive, enabling structured alert creation, promotion to investigation cases, Cortex observable analysis, and documented analyst response.
- Developed JSON mappings, webhook integrations, alert metadata, case workflows, and threat-intelligence enrichment for SOC incident handling.
